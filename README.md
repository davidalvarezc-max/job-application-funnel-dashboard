# Job Application Funnel Analysis Dashboard (Power BI)

## Overview
This project analyzes the job application process using a sales funnel approach. The dashboard provides insights into application volume, conversion rates, and bottlenecks across different stages of the hiring pipeline.

## Objective
Treat the job search process as a sales pipeline and analyze application activity to identify conversion rates, bottlenecks, and trends. The goal is to use data to improve the job search strategy and increase the probability of securing interviews and offers.

## Dataset
The dataset was created from a personal job application tracker. It includes the application date, job source (portal), role applied for, priority indicator for strategic opportunities, and the current stage of the hiring process.

This structure allows analysis of application rhythm, evaluation of the job search strategy, and measurement of conversion rates across the pipeline.

## Tools & Methodology
- Microsoft Excel (data collection)
- Power BI
- Data Modeling
- DAX Measures
- Data Visualization

## Key Insights
- The majority of applications remain in early pipeline stages, highlighting the importance of improving screening and interview conversion rates.
- Certain job portals generate higher application volume than others.
- Tracking applications over time helps maintain a consistent job search rhythm and identify periods of higher activity.
